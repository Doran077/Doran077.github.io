Vue的使用

在js代码中使用Vue时，先构造一个vue对象的实例。

Vue是个构造函数，该构造函数在创建实例时可向内部传入一个对象作为参数，对象内部可存放的属性有：

el：其值为css选择器，表示该vue实例管理的DOM区域

data：其值为一个对象或一个函数，内部存储的是该管理DOM区域的数据

methods：其值为一个对象，内部存储函数(事件监听的回调)
注意：这些函数中如果使用了this则不能使用箭头函数。

Vue生命周期详解
